# A Refined Mind 🧠

A clean, modern, and emotionally resonant personal portfolio website centered on faith-driven storytelling, spiritual growth, and life lessons.

## Brand Essence

"A Refined Mind" is a reflective, faith-driven personal brand that documents personal experiences, extracts meaning from them, and glorifies God through storytelling and insight.

**Voice:**
- Thoughtful
- Introspective
- Spiritually grounded (Christian perspective)
- Honest, not overly polished

## Features

### Pages
- **Home** - Hero section with tagline, featured writings, and key themes
- **About** - Personal story and faith journey
- **Writings** - Blog-style layout with category filtering
- **Single Post** - Distraction-free reading experience
- **Contact** - Simple connection page

### Design
- **Colors:** White and Blue brand palette
- **Typography:** Clean sans-serif fonts
- **Layout:** Minimalist with lots of whitespace
- **Experience:** Smooth scrolling and subtle animations

### Content Categories
- Faith & Spiritual Growth
- Life Lessons
- Personal Reflections
- Discipline & Mindset
- Purpose & Calling

## Getting Started

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

## Tech Stack

- **React** - UI framework
- **React Router** - Navigation
- **Vite** - Build tool
- **CSS** - Styling

## Project Structure

```
src/
├── components/
│   ├── Layout.jsx       # Main layout with navigation
│   └── Layout.css
├── pages/
│   ├── Home.jsx         # Homepage
│   ├── About.jsx        # About page
│   ├── Writings.jsx     # Blog listing
│   ├── Post.jsx         # Single post view
│   └── Contact.jsx      # Contact page
├── App.jsx              # Main app component
├── main.jsx             # Entry point
└── index.css            # Global styles
```

## Content Guidelines

- Write in first person
- Avoid clichés and overly preachy tone
- Use storytelling structure: Experience → Reflection → Insight → Spiritual takeaway
- Keep it real — struggles, growth, lessons
